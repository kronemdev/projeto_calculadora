🧮 Projeto Calculadora (UI Design)
Este é um projeto de uma interface de calculadora desenvolvido para praticar conceitos fundamentais de HTML5 e CSS3. O foco principal foi a criação de um layout moderno, responsivo e visualmente atraente, utilizando técnicas avançadas de posicionamento.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica dos botões e visor.
CSS3: Estilização completa, incluindo:
CSS Grid: Para organizar as teclas de forma precisa.
Flexbox: Para centralização e alinhamento de itens.
Design Responsivo: Ajustes para garantir que a calculadora fique bonita em computadores e celulares.
Efeitos de Hover: Interatividade visual ao passar o mouse nos botões.

🎨 O que eu aprendi
Neste projeto, pude aprofundar meus conhecimentos em:
Organização de containers e hierarquia visual.
Uso de sombras (box-shadow) e bordas arredondadas para dar profundidade.
Criação de classes específicas para diferentes tipos de botões (operadores, números e ações).
